A modern, minimalist Counter App built with a focus on clean UI/UX and Glassmorphism design principles. This project was developed as a lightweight utility to demonstrate frontend fundamental skills.

**Features**
Real-time Interaction: Instant UI updates using JavaScript.

Modern UI: Glassmorphism card effect with CSS Backdrop-filter.

Responsive Design: Fully functional on mobile, tablet, and desktop screens.

Tactile Feedback: Animated button interactions (hover and active states).

Reset Functionality: Easily clear progress with a single click.

** Tech Stack**
HTML5: Semantic structure.

CSS3: Custom animations, Flexbox layout, and Glassmorphism
